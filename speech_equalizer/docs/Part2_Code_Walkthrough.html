2.1 Initialization & Audio Input

fname = input('Audio file path: ','s');
[x_orig, fs_orig] = audioread(fname);
if size(x_orig,2) > 1, x_orig = mean(x_orig,2); end
  • audioread(fname) reads a .wav file. It returns x_orig (the audio data matrix, values -1 to 1) and fs_orig (the sampling rateSampling Rate (fs)How many times per second the microphone measured the air pressure. Usually 44,100 Hz for CD quality audio.).
  • if size > 1 checks if the audio is stereo (2 channels). If so, mean(x_orig, 2) averages the left and right channels into a single mono signal for processing.

2.2 Filter Design Loop (The Core)

This is where the math happens. We design one filter for each frequency band.

nyq = fs_orig / 2;
b_all = cell(1,nbands);
a_all = cell(1,nbands);

for i = 1:nbands
    fl = bands(i,1); fh = min(bands(i,2), nyq-1);
    
    % Convert to normalized frequency
    Wn = [max(fl/nyq,0.001) fh/nyq]; ft = 'bandpass';
    
    if ftype == 1 % FIR
        w = hamming(filt_order+1);
        b_all{i} = fir1(filt_order, Wn, ft, w);
        a_all{i} = 1;
    else % IIR
        [b_all{i},a_all{i}] = butter(filt_order, Wn, ft);
    end
end
  • nyq = fs_orig / 2 calculates the Nyquist FrequencyNyquist FrequencyThe absolute maximum frequency your digital signal can contain. All MATLAB filters require you to define your cutoffs as a fraction (0 to 1.0) of this Nyquist value..
  • Wn = [fl/nyq fh/nyq] converts the user's raw Hz edges (e.g., 100 to 300 Hz) into a normalized fraction required by MATLAB's filter design algorithms.
  • FIR Path: Uses fir1 along with a Hamming windowHamming WindowA math function that tapers the edges of the filter coefficients to zero. This prevents the "Gibbs Phenomenon," which would otherwise cause the frequency response to ripple and ring wildly. to create a highly precise, linear-phase bandpass filter. a_all{i} = 1 because FIR filters have no feedback loop.
  • IIR Path: Uses butter (Butterworth) to generate both feedforward (b) and feedback (a) coefficients.

2.3 Signal Processing — Filtering & Gain

y_bands = zeros(length(x), nbands);

for i = 1:nbands
    if ftype == 1
        yi = filter(b_all{i}, a_all{i}, x);
    else
        yi = filtfilt(b_all{i}, a_all{i}, x);
    end
    
    % Apply Volume Gain
    gain_lin = 10^(gains_db(i)/20);
    y_bands(:,i) = yi * gain_lin;
end

% Recombine the signal
y_eq = sum(y_bands, 2);
y_eq = y_eq / max(abs(y_eq)+eps);
  • filter() processes the audio normally. Used for FIR because FIR naturally preserves the waveform shape.
  • filtfilt() is the Zero-Phase trickZero-Phase FilteringIIR filters naturally distort the phase (time-alignment) of speech. `filtfilt` runs the audio forward through the filter, then reverses it and runs it backward. The two distortions perfectly cancel each other out! used for IIR to prevent it from smearing the speech transients.
  • 10^(gains_db(i)/20) converts the decibel gainDecibels (dB)A logarithmic scale for loudness. +6 dB means the linear multiplier is roughly 2.0 (twice the amplitude). -6 dB means the linear multiplier is roughly 0.5 (half amplitude). the user typed into a standard linear multiplier.
  • sum(y_bands, 2): After every band has been isolated and gained, we mathematically add them all back together. Because our bands cover the whole spectrum (0 to 20kHz) without gaps, adding them up perfectly reconstructs the original audio—just with the user's EQ boosts applied!

2.4 Sample Rate Conversion

y_4x = resample(y_eq, 4, 1);
y_half = resample(y_eq, 1, 2);
  • resample(signal, 4, 1) performs upsamplingUpsamplingIncreases the sample rate by 4x. MATLAB automatically inserts zeros and applies a low-pass anti-imaging filter to smoothly interpolate the missing data points..
  • resample(signal, 1, 2) performs downsamplingDownsamplingCuts the sample rate in half. MATLAB automatically applies an anti-aliasing low-pass filter BEFORE discarding every other sample, preventing high-frequency noise from corrupting the lower frequencies..
